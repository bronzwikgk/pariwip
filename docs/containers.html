Containers

Containers are made to hold elements, components, data, and even containers also.


These are empty templates to put in data.


There are various types of containers in pari:


Block Container

This is some block container
                    

<div class="container-block"> 
    This is some block container
    <!-- Put your content here -->
</div>

Float Container

This is some floating container
                    

<div class="container-float"> 
    This is some floating container
    <!-- Put your content here -->
</div>

Flex Container

Item1 Item2 Item3 Item4 
                        

<div class="container-flex" style="max-width: 100%;"> 
    This is some flex container
    <!-- Put your content here -->
</div>

Pop Container

This is some pop container (u can drag it). 
                    

<div class="container-pop"> 
    This is some pop container (u can drag it).
    <!-- Put your content here -->
</div>

Resizable Container

This is some Resizable container
                    

<div class="container-resizable"> 
    This is some Resizable container
    <!-- Put your content here -->
</div>

Collapsible Container

This is some collapse collapse-top container
                    

<div class="container-collapsible collapse-top"> 
    This is some collapsible collapse-top container
    <!-- Put your content here -->
</div>

Stacking Container

This is some stack container 1
This is some stack container 2
This is some stack container 3 
                    

<div class="container-stack" data-stack-index="1"> This is some stack container 1 </div>
<div class="container-stack" data-stack-index="2"> This is some stack container 2 </div>
<div class="container-stack" data-stack-index="3"> This is some stack container 3 </div>
