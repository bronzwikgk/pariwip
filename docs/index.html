Introduction

Get familiar with the basic setup and overview of Pari CSSDownload
Package contents:The Zip file contains the compiled CSS and JavaScript files, which is everything you need to get started
Folder Description
/css Contains the Pari's CSS Files
/js Contains the Pari's JavaScript Files
Html MarkupAdd the compiled and minified CSS and JavaScript to the head element of your HTML5 document. For your basic setup, that's it.
Example:
                        
                        
<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Title</title>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <link rel="stylesheet" href="css/pari.css" />
        <link rel="stylesheet" href="css/light-theme.css" /> <!-- Replace this with your desired theme file -->
        <script src="js/pari.js"></script>
    </head>
    <body>
        <!-- Your work goes here -->
    </body>
</html>

Build From Source

Optionally, you can build Pari CSS from its source on Github. To do this, you'll need to have Node.js and npm installed. We use pure CSS and Vanilla JS to build Pari CSS.

  
  git clone https://github.com/bronzwikgk/pari.live.git
  cd pari
  npm install
  npm start
